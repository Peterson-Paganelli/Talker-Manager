<h1>Talker-manager</h1>
<h1>Descrição</h1>
<div>
  <p>Projeto desenvolvido na Trybe durante o módulo de Back-End</p>
  <p>Nesse projeto fizemos uma API de um CRUD para cadastro de palestrantes e

  </p>
</div>
